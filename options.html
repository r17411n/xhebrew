Google Cloud Translate (optional)

If enabled, translations will use the official Google Cloud Translate endpoint. Storing an API key in extension storage is convenient but not perfectly secret — consider using a background proxy for production.

Use standard BCP-47 language codes (e.g. en, fr). This feature sends text to Google's translate endpoint; it may be rate limited.

Use `/pattern/flags` for regex matches (e.g. /שלום/gi), or plain text for literal matches.
